Microsoft Share Point:

SharePoint или Microsoft SharePoint Products and Technologies – набор приложений и технологий корпоративного класса, дающих возможность эффективно решать такие задачи бизнеса, как управление информацией и текущими бизнес-процессами, организация быстрого поиска сведений и обмена ими между пользователями:

Windows SharePoint Services (WSS) – универсальная технология, предоставляющая инфраструктуру для совместной работы. WSS является бесплатным приложением для Windows Server. До этого WSS был известен под названием SharePoint Team Services. Но после выхода версии WSS 4.0 Microsoft переименовала приложение в SharePoint Foundation 2010.

Microsoft Office SharePoint Server (MOSS) — интегрированный набор серверных приложений (SharePoint сервер), помогающих сотрудникам и группам сотрудников организовывать совместную работу . MOSS является платным компонентом для осуществления взаимодействия функциональности SharePoint и приложений MS Office. MOSS так же является надстройкой над WSS, тем самым способствует расширению его возможностей. Ранее носил название SharePoint Server и SharePoint Portal Server;

Microsoft Office SharePoint Designer (SPD) – инструмент для настройки и создания мощных приложений на платформе SharePoint. SPD — HTML-редактор, используемый для создания SharePoint-страниц и управления документами для WSS сайтов. С весны 2009 г. SPD стал бесплатным продуктом.

Объединяя возможности совместного использования Office SharePoint Server и SharePoint Foundation (ранее Windows SharePoint Services), а также разработки и настройки Office SharePoint Designer 2010 работники имеют возможность создавать, разрабатывать и контролировать личные узлы SharePoint, к которым так же могут присоединяться и другие пользователи организации.

Microsoft SharePoint


Microsoft Office SharePoint Server – это серверная программа, предоставляющая дополнительные возможности по совместной работе, управлению информацией, реализации бизнес-процессов и обеспечению доступа к данным, очень важным для задач компании и ее работы.

SharePoint помогает компаниям результативно организовывать корпоративную информацию, предоставляет быстрый доступ работников к ней, реализовывает бизнес-процессы, управляет совместной работой и документами сотрудников, использует и распределяет информацию как внутри, так и вне организации. Microsoft SharePoint это интегрированная платформа, содержащая в себе рабочие области групп, порталы, уведомления, электронную почту, быстрые сообщения, веб-конференции – эффективное решение для коллективной работы, доступное пользователям через известные и удобные MS Office приложения и веб-браузер внутри локальной сети и через Интернет.




Microsoft SharePoint Server (MS SharePoint) входит в состав комплекта Microsoft Office System, в основе которого лежит Microsoft SharePoint Foundation (ранее Windows SharePoint Services) — базовая технология, обеспечивающая производительность и взаимодействие рабочих групп за счет предоставления доступа к людям, документам и данным, необходимым для эффективного выполнения работы и для принятия обоснованных решений, важных для компании. Все возможности, доступные в Microsoft SharePoint Foundation, также доступны в SharePoint Server.
